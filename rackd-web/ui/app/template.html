Gateway
None

On this mode no router will respond to Router Solicitations (RS) nor send Router Advertisements (RA), meaning that end-user devices will need to manually configure their default gateway if they want to communicate with hosts outside their local network.

VRRP

The Virtual Router Redundancy Protocol (VRRP) defined in RFC5798 provides redudancy for the default gateway. If this option is enabled all routers in the LAN will become part of a single VRRP Group.

✔

On this mode a single router acts as the default gateway. This mode creates a Single Point of Failure and should only be used when there is a single node in the network.

IPoE

IPoE is plain IP over Ethernet, Hosts connecting to IPoE Service Providers are typically configured dynamically via DHCP, or albeit less common, can be statically configured.

PPPoE

PPPoE encapsulates PPP frames over Ethernet to create a Point-To-Point link between the CPE and the router's PPPoE Server. Use of PPPoE is falling out of favor due to the higher overhead compared to IPoE.
